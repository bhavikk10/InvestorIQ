# Investor Automation Demo

A modern React-based dashboard application built with Vite, featuring analytics, project management, and reporting capabilities.

## Getting Started

### Prerequisites

- Node.js (v18 or higher recommended)
- npm or yarn package manager

### Installation

1. Clone the repository:
```bash
git clone <repository-url>
cd investorautomationdemo
```

2. Install dependencies:
```bash
npm install
```

### Running the Application

To start the development server:
```bash
npm run dev
```

The application will be available at `http://localhost:5173` (or the port shown in your terminal).

### Building for Production

To create a production build:
```bash
npm run build
```

To preview the production build locally:
```bash
npm run preview
```

### Other Commands

- Run ESLint:
```bash
npm run lint
```

## Tech Stack

- React 19
- Vite
- React Router DOM
- Tailwind CSS
- Recharts
- Framer Motion
- Lucide React Icons
